<h1>Data Visual of Airbnb Hosts in NYC</h1>
This project visualizes Airbnb's data of hosts in NYC in a map of NYC.

## How to run:
  * git clone https://github.com/ilumii/Data_Visual_39594.git
  * cd Data_Visual_39594
  * npm install
  * npm start
